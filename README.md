# skill-industry-research
专业的产业研究与战略分析技能，为投资决策和商业分析提供深度洞察。

## 📦 包含技能

### industry-research - 产业研究专家

**功能定位**: 资深的产业研究专家与战略顾问

**核心理念**: 
- 🎯 决策导向 - 为不同角色提供可落地的行动建议
- 🔍 深度洞察 - 解释"为什么"和"意味着什么"
- 📊 用已知推断未知 - 利用代理指标推导难以获取的数据
- 🌐 系统化分析 - 横向×纵向的分析矩阵
- ✅ 数据交叉验证 - 多维测算方法

**研究框架**:
1. 执行摘要 - 核心洞察和关键数据
2. 产业深度定义 - 本质解析和边界界定
3. 产业链拆解 - 价值链分析和参与者图谱
4. 市场规模测算 - 自上而下/自下而上测算
5. 竞争格局分析 - 波特五力模型
6. 增长驱动力 - 需求/供给/政策/技术驱动
7. 风险因素 - 周期性/竞争/政策/技术风险
8. 趋势研判 - 短期/中期/长期趋势
9. 投资建议 - 针对不同角色的建议

## 🚀 安装使用

### 方式一：直接复制

```powershell
# Windows
xcopy /E /I "industry-research" "%USERPROFILE%\.config\opencode\skills\industry-research"
```

### 方式二：符号链接

```powershell
$repoDir = "C:\path\to\opencode-business-research"
$skillsDir = "$env:USERPROFILE\.config\opencode\skills"
New-Item -ItemType SymbolicLink -Path "$skillsDir\industry-research" -Target "$repoDir\industry-research"
```

## 📋 使用示例

### 示例 1：新能源汽车行业研究

```
请对新能源汽车行业进行深度研究分析，包括：
1. 行业市场规模和增长趋势
2. 产业链关键环节分析
3. 主要竞争对手和竞争格局
4. 投资机会和风险提示
```

```
请分析当前半导体行业的投资机会：
- 上游设备和材料
- 中游晶圆制造
- 下游封装测试
需要包括政策支持、技术突破和市场需求分析
```

## 📊 研究方法论

### 分析维度

| 维度 | 分析内容 | 工具方法 |
|------|----------|----------|
| 政策 | 产业政策、监管环境 | PEST分析 |
| 市场 | 市场规模、增长率 | TAM/SAM/SOM |
| 技术 | 技术路线、创新趋势 | 技术成熟度曲线 |
| 产业链 | 上下游关系、议价能力 | 价值链分析 |
| 竞争 | 竞争格局、护城河 | 波特五力模型 |

### 数据来源

- 官方统计数据
- 行业协会报告
- 上市公司财报
- 专家访谈
- 实地调研
